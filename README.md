# Pham Tien Dat - Mechanical Engineering Portfolio

Static portfolio website for internship applications.

## Run locally
Open `index.html` directly, or use a static server.

## Project structure
- `index.html`: main website
- `css/`, `js/`, `images/`: frontend assets
- `assets/models`: 3D files
- `assets/documents`: PDF files
- `assets/cad`: CAD handoff files
- `assets/downloads`: packaged ZIP files for one-click download

## Deploy
Use `DEPLOY.md` for step-by-step deployment on GitHub Pages or Netlify.

## Important post-deploy task
After you have public URL, update:
- `robots.txt`
- `sitemap.xml`
Replace `https://your-domain.com` with your real domain.
